Space Runner is a Geometry Dash inspired platformer sidescroller. 

Watch [this video](https://youtu.be/Km0s_ZYAVu0) to see the gameplay!

View [the paper](https://github.com/ShelbyPop/SpaceRunner/blob/main/Space%20Runner%20Paper.pdf) to see how its implemented without having to dive into the code.
